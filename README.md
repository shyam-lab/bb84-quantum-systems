# 🔐 BB84 Quantum Key Distribution — Systems Exploration

This project explores BB84 from a systems perspective, focusing on:

- Noise models (bit-flip vs depolarizing)
- QBER analysis
- Eavesdropping (intercept-resend)
- Statistical security behavior

## Key Insight

Not all noise is equally detectable. Structured errors are easier to detect than random noise.
